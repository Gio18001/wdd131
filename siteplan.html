Siteplan

Site Name

A currently available web domain that would be good for this site is: pseudoregaliainfo.com

Site Purpose

This site will contain information about the game Pseudoregalia, an indie metroidvania game by Rittzler. It will contain some history of the game and its development, screenshots and maps of the game itself, and a page dedicated to advanced movement techniques not fully explained in the game. It will also have links to the itch.io and steam store pages for the game, so anyone viewing it can easily check out the game themselves.

Scenarios

Some questions one may be asking when visiting the site:

Where did Pseudoregalia come from?

There seems to be unexplained controls, what are those?

What more can I know about the game before I spend money to try it?
